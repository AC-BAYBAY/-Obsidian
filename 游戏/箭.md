# 基本作用
传送箭（仅激活机关）
打破气球（掉落道具，砸石头）
站立（开场有个小怪站在箭上）
击杀小怪
# 穿透区域
**基础穿透箭**  
箭可穿透藤蔓/薄墙，射中后面的机关。教学关教“箭可穿透弱点”，后续需射穿多层障碍命中目标。

**火焰穿越**：箭穿过火焰变为“火箭”，可点燃藤蔓、油桶或火炬。教学关用明显的蜡烛阵引入，后续需用一道火墙同时点燃多根远程引线。

冰**霜穿越**：穿过寒气或冰雾变为“冰箭”，能冻结水面、齿轮或敌人。先在有标记的冷雾区学会冻结落下的水滴搭桥，之后需要让箭穿过移动的冷气团。

**雷电充能**：穿过电圈或电云获得电流，可激活磁力设备、导通电路。教学关给静止电环，后期需在电极间跳跃的电流束中精准穿过。

 ### **磁性附着**：穿过磁场，箭暂时带磁，可吸附在金属表面并传递磁力。教学关用来吸起小金属钥匙，后续需在复杂电路板上通过吸附接通断路。

**冻结路径**：穿过极寒冰核，箭飞行尾迹凝结出冰轨（碰到蒸汽），可让人滑行或引导物体滑动。先滑过深渊，后需让冰块沿冰轨滑向目标。
# **元素反应**    
如火+冰=蒸汽云雾（遮蔽视线但可触发蒸汽机），电+水=范围电击。先教火+油=爆炸，后需依次穿越不同云雾合成正确效果破解多层防御。

# 旋转风扇
吹风改变箭的方向，吹动水蒸气、火焰
# 脆弱支柱
击断承重支柱，使上方的平台或重物坍塌，重构地形。可用于砸开下方薄弱地板或压住按钮。（击杀怪物）
# 平衡跳板
击中跷板一端（将东西打落），将另一端上的物体或主角弹起。可用于投掷钥匙或抵达高台。
