# Obsidian Windows 常用快捷键

## 通用与文件操作
| 功能            | 快捷键                |
| ------------- | ------------------ |
| 新建笔记          | `Ctrl + N`         |
| 快速切换笔记        | `Ctrl + O`         |
| 打开命令面板        | `Ctrl + P`         |
| 保存当前笔记        | `Ctrl + S`         |
| 打开设置          | `Ctrl + ,`         |


## 文本编辑
| 功能       | 快捷键                             |
| -------- | ------------------------------- |
| 加粗       | `Ctrl + B`                      |
| 斜体       | `Ctrl + I`                      |
| 撤销       | `Ctrl + Z`                      |
| 在下方新增行   | `Ctrl + Shift + Enter`          |
| 小标题      | `Ctrl + Shift + Z`              |
| 1~X级标题   | `Ctrl + 1~X`                    |
| 引用       | `Ctrl + Shift + C`              |
| 重做       | `Ctrl + Shift + Z` 或 `Ctrl + Y` |
| 全选       | `Ctrl + A`                      |
| 查找（当前笔记） | `Ctrl + F`                      |
| 替换（当前笔记） | `Ctrl + H`                      |

## 视图与导航
| 功能 | 快捷键 |
|------|--------|
| 切换编辑 / 阅读视图 | `Ctrl + E` |
| 后退 | `Ctrl + Alt + ←` |
| 前进 | `Ctrl + Alt + →` |
| 在编辑器中跟随链接 | `Ctrl + Enter` |
| 全局搜索（所有文件） | `Ctrl + Shift + F` |
| 打开关系图谱 | `Ctrl + G` |
| 插入模板 | `Ctrl + T` |
| 切换左侧边栏 | `Ctrl + Shift + I` |
| 切换右侧边栏 | `Ctrl + Shift + R` |
| 打开文件浏览器 | `Ctrl + Shift + E` |
| 垂直拆分窗格 | `Ctrl + \` |
| 放大界面 | `Ctrl + 加号` 或 `Ctrl + 鼠标滚轮` |
| 缩小界面 | `Ctrl + 减号` 或 `Ctrl + 鼠标滚轮` |
| 重置界面缩放 | `Ctrl + 0` |
| 打开开发者工具 | `Ctrl + Shift + Alt + I` |

## 窗格与焦点
| 功能 | 快捷键 |
|------|--------|
| 将焦点移回编辑区 | `Esc`（当焦点在侧边栏时） |
