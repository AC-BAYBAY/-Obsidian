
## 📁 文件操作
| 功能             | 快捷键              |
|----------------|-------------------|
| 新建文件         | `Ctrl + N`        |
| 保存当前文件     | `Ctrl + S`        |
| 另存为新文件     | `Ctrl + Shift + S`|
| 修改画布大小     | `Ctrl + Alt + C`  |

## 🧩 图层与分组
| 功能                   | 快捷键                |
|----------------------|---------------------|
| 新建图层               | `Shift + N`         |
| 新建组                 | `Alt + Shift + N`   |
| 复制当前帧/Cel         | `Alt + D`           |
| 隐藏/显示当前图层       | `Shift + X`         |
| 展开/折叠图层组         | `Shift + E`         |
| 图层不透明度 10%~90%    | `1` ~ `9`（数字键）  |
| 图层不透明度 100%       | `Shift + 4`         |
| 切换前景色/背景色       | `X`                 |
| 重置默认黑白（前黑后白） | `D`                 |

## 🧠 动作修改器（操作时额外按键）
| 效果                       | 操作方式                              |
|--------------------------|-------------------------------------|
| 画直线（铅笔）             | 按住 `Shift`                        |
| 锁定移动轴                | 移动时按住 `Shift`（按 X 或 Y 轴锁定）  |
| 绘制正圆/正方形            | 使用矩形/椭圆工具时按住 `Shift`         |
| 从中心绘制图形             | 使用矩形/椭圆工具时按住 `Ctrl`          |
| 复制选区并移动            | 移动前按住 `Ctrl`                    |
| 自动选择图层（移动工具下） | 按住 `Ctrl` ＋ 点击画布中的图像         |

## ⚙️ 自定义快捷键
| 功能                 | 快捷键                    |
|--------------------|-------------------------|
| 打开快捷键设置界面   | `Ctrl + Alt + Shift + K` |
| 打开偏好设置         | `Ctrl + K`              |

> 💡 **提示**：在菜单栏 `编辑 > 键盘快捷键` 中可以搜索并修改任意功能的快捷键组合。
